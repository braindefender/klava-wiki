# LED подсветка

В клавиатурах обычно выделяют два типа подсветки:

![](/assets/hardware/led/underglow.jpg)
**underglow** — подсветка снизу корпуса/платы

![](/assets/hardware/led/backlight.jpg)
**backlight** — подсветка [кейкапов](./keycaps.md)

Для всех типов подсветки в клавиатурных прошивках есть множество различных эффектов. Можно настроить так, чтобы подсветка менялась в зависимости от текущего слоя.
